# Mejoras visuales v2

Esta versión aplica únicamente cambios de presentación en el proyecto web.

## Archivos modificados

- `ClinicaDental.Web/Pages/Auth/Login.cshtml`
- `ClinicaDental.Web/Pages/Index.cshtml`
- `ClinicaDental.Web/Pages/Shared/_Layout.cshtml`
- `ClinicaDental.Web/wwwroot/css/site.css`

## Cambios aplicados

- Login con diseño más moderno tipo glassmorphism.
- Fondo animado con orbes suaves.
- Tarjeta de login con microanimaciones.
- Botón de login con transición visual.
- Página principal más limpia y menos cargada.
- Accesos rápidos a Citas, Pacientes, Inventario y Ventas.
- Animaciones suaves de entrada para páginas y tarjetas.
- Respeto a `prefers-reduced-motion` para reducir animaciones si el usuario lo tiene configurado.

## Funcionalidad

No se modificaron controladores, modelos, servicios, repositorios, base de datos ni archivos `.cshtml.cs`.
